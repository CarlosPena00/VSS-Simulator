**Install** Ubuntu (16.04 LTS)
------------------------------
**Install Standard Packages:**
```
$ sudo apt-get update && apt-get upgrade
$ sudo apt-get install pkg-config
$ sudo apt-get install g++ cmake git libzmq3 libzmq3-dev protobuf-compiler libprotobuf-dev freeglut3 freeglut3-dev libboost-all-dev
```
